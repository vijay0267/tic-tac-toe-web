# 🎮 Tic-Tac-Toe Web

A single-player Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play against an AI opponent right in your browser.

## ▶️ Play Now

**[Click here to play!](https://vijay0267.github.io/tic-tac-toe-web/)**

## Features

- 🧠 Smart AI opponent — tries to win, blocks your moves
- ✨ Animated X and O marks with pop-in effects
- 🏆 Scoreboard to track wins, losses, and draws
- 🌟 Winning line highlights with golden glow
- 📱 Responsive design — works on desktop and mobile
- 🎨 Modern dark gradient theme

## How to Play

1. You play as **X**, AI plays as **O**
2. Click any empty cell to place your mark
3. Get three in a row (horizontal, vertical, or diagonal) to win
4. Click **🔄 New Game** to start a new round

## Tech Stack

- HTML5
- CSS3 (animations, gradients, grid layout)
- Vanilla JavaScript (no frameworks)
